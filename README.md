🎮 Tic-Tac-Toe AI (GUI Mini Project) 📌 Overview

A GUI-based Tic-Tac-Toe game where a user plays against an AI. The AI makes smart moves using a rule-based strategy (win, block, best position).

⚙️ Technologies Python Tkinter (GUI)

🧠 Working User plays as X, AI plays as O AI logic: Try to win Block opponent Choose best available position Game detects win or draw automatically

🚀 How to Run Save as tictactoe.py Run: python tictactoe.py Click cells to play

✨ Features Interactive GUI Smart AI opponent Win highlighting & animations Reset option

👨‍💻 Conclusion

Demonstrates basic AI decision-making and GUI development through an interactive Tic-Tac-Toe game.
